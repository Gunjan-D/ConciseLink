ShortLink API Documentation

Welcome to the ShortLink API! Our REST API allows you to programmatically create, manage, and track shortened URLs. This documentation will help you integrate ShortLink into your applications.

Base URL

Base URL 
https://api.shortlink.com/v1

Features

  • ✅ Create and manage short URLs
  • ✅ Advanced analytics and tracking
  • ✅ Custom short codes and domains
  • ✅ Bulk operations
  • ✅ Real-time webhooks
  • ✅ Password protection and expiration
  • ✅ QR code generation

Authentication

ShortLink API uses API key authentication. Include your API key in the Authorization header of each request.

API Key Authentication

Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY

Getting Your API Key

  1. Sign up for a ShortLink account
  2. Go to your dashboard settings
  3. Navigate to the "API Keys" section
  4. Generate a new API key
  5. Copy and securely store your API key

Rate Limiting

To ensure fair usage and maintain service quality, our API implements rate limiting.

Rate Limits

  • Free Plan: 1,000 requests per hour
  • Pro Plan: 10,000 requests per hour
  • Enterprise Plan: 100,000 requests per hour

Rate Limit Headers

Header Description Example
X-RateLimit-Limit The rate limit ceiling for that given request 1000
X-RateLimit-Remaining The number of requests left for the time window 999
X-RateLimit-Reset UTC epoch seconds when the rate limit resets 1640995200

Create Short URL

POST /urls
Create a new shortened URL with optional customization options.

Request Body Parameters

Parameter Type Required Description
original_url string Required The original URL to be shortened
custom_short_code string Optional Custom short code (3-50 characters)
expires_at datetime Optional ISO 8601 expiration date
title string Optional Title for the link (max 200 characters)
password string Optional Password protection for the link
tags array Optional Array of tags for organization 
curl -X POST \
  https://api.shortlink.com/v1/urls \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "original_url": "https://example.com/very-long-url",
    "custom_short_code": "my-link",
    "title": "My Custom Link",
    "expires_at": "2024-12-31T23:59:59Z",
    "tags": ["marketing", "campaign"]
  }'
const createUrl = async () => {
  const response = await fetch('https://api.shortlink.com/v1/urls', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      original_url: 'https://example.com/very-long-url',
      custom_short_code: 'my-link',
      title: 'My Custom Link',
      expires_at: '2024-12-31T23:59:59Z',
      tags: ['marketing', 'campaign']
    })
  });
  
  const data = await response.json();
  console.log(data);
};

URL Analytics

GET /analytics/{short_code}
Get detailed analytics for a specific short URL.

Query Parameters

Parameter Type Required Description
period string Optional Time period: 1h, 24h, 7d, 30d, 90d, all (default: 30d)
breakdown string Optional Data breakdown: hour, day, week, month (default: day) 
curl -X GET \
  'https://api.shortlink.com/v1/analytics/my-link?period=7d&breakdown=day' \
  -H 'Authorization: Bearer YOUR_API_KEY'
const getAnalytics = async (shortCode) => {
  const response = await fetch(
    `https://api.shortlink.com/v1/analytics/${shortCode}?period=7d&breakdown=day`,
    {
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY'
      }
    }
  );
  
  const analytics = await response.json();
  console.log(analytics);
};
import requests

short_code = "my-link"
url = f"https://api.shortlink.com/v1/analytics/{short_code}"
params = {"period": "7d", "breakdown": "day"}
headers = {'Authorization': 'Bearer YOUR_API_KEY'}

response = requests.get(url, params=params, headers=headers)
analytics = response.json()

Python SDK Example

Python SDK Usage 
from shortlink import ShortLinkClient

# Initialize client
client = ShortLinkClient(api_key='YOUR_API_KEY')

# Create a short URL
url = client.urls.create(
    original_url='https://example.com/very-long-url',
    custom_short_code='my-link',
    title='My Custom Link'
)

print(f"Short URL: {url.short_url}")

# Get analytics
analytics = client.analytics.get('my-link', period='7d')
print(f"Total clicks: {analytics.total_clicks}")

# List all URLs
urls = client.urls.list(limit=50)
for url in urls:
    print(f"{url.short_code}: {url.click_count} clicks")

Error Handling

The ShortLink API uses standard HTTP response codes to indicate success or failure of requests.

Error Response Format

Error Response 
{
  "error": {
    "code": "INVALID_URL",
    "message": "The provided URL is not valid",
    "details": {
      "field": "original_url",
      "reason": "Invalid URL format"
    }
  },
  "request_id": "req_1234567890abcdef"
}

Common Error Codes

HTTP Code Error Code Description
400 INVALID_URL The provided URL is not valid
400 SHORT_CODE_TAKEN The custom short code is already in use
401 INVALID_API_KEY The API key is invalid or missing
404 URL_NOT_FOUND The requested URL was not found
429 RATE_LIMIT_EXCEEDED Too many requests, rate limit exceeded
500 INTERNAL_ERROR An internal server error occurred
